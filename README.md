# Wireframe → ASCII

A minimalist wireframing tool that draws UI layouts and exports them as ASCII art.

## Features

- **Visual Wireframing** - Draw rectangles, placeholders, arrows, lines, and text labels on a grid canvas
- **Live ASCII Output** - See the ASCII representation update in real-time as you draw
- **Export Options** - Copy as plain ASCII or wrapped in an AI-friendly prompt
- **Keyboard Shortcuts** - Quick access to all tools

## Tools

| Tool | Shortcut | Description |
|------|----------|-------------|
| Select | V | Click to select, drag to move, double-click to edit label |
| Rectangle | R | Drag to draw boxes for panels, cards, modals, sidebars |
| Placeholder | P | Drag to create dashed placeholders for images, avatars, charts |
| Arrow | A | L-shaped arrows showing flow direction |
| Line | L | Plain lines for dividers (hold Shift to snap) |
| Text | T | Click to place standalone text labels |


## Keyboard Shortcuts

- `V` - Select tool
- `R` - Rectangle tool
- `P` - Placeholder tool
- `A` - Arrow tool
- `L` - Line tool
- `T` - Text tool
- `?` - Toggle help
- `+` / `-` - Zoom in/out
- `0` - Reset zoom
- `Cmd+Z` - Undo
- `Delete` - Delete selected element
